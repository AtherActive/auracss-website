Quickstart guide

Want to get going quick? We got you! To start, you'll have o do a few things:

  1. Download AuraCSS from out Github Repo. If you use the CDN, skip to step [].
  2. Make sure the auracss-[version].css is in your project directory.
  3. Create a new HTML file. For this guide we'll call it hello.html.
