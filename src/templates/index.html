Keep it simple

This small CSS framework has a suprising amount of features for it's size. Want colums? Check! Maybe you want images on your background, essentially small banners? We got that too! Everything can easily be configured from either the primary sheet or a secondary sheet (both are available for download)

Features

Grid

It would not be a powerful framework without some way to have a grid!

Styles

By default, you can adjust the framework to use 2 styles!

Responsive

Of course AuraCSS is fully responsive. From the colums to the navbar, everything will adjust for small devices.

Background Image Support

Aything can have an image as background, the easy way!

Easy configuration

Everything is easily configurable! Most things can be controlled by Vars and regular CSS.

Built-in Navigation

Navbars can be hard, but they do not have to be! Using the UL approach you can make a navbar in minutes!

Why use AuraCSS?

It's all about the right tool for the job. AuraCSS is a very simple framework, and while it does have a ton of features, it is not designed to be run on supermassive sites like social media platforms. Of course, it is completely up to you!


This website was made completely and only with AuraCSS, with only 2 additional stylesheets: A small custom one to modify AuraCSS for coloring and the Boxicon sheet for icons.
